# homebrew-tap

Homebrew tap for [polius](https://github.com/polius) tools.

## Install

```bash
brew install polius/tap/fsend
```

`fsend` — peer-to-peer, end-to-end encrypted file transfer ([repo](https://github.com/polius/fsend)).

## Notes

- macOS only (Homebrew casks). Linux users: see each tool's own install instructions.
- Cask files under `Casks/` are generated and updated automatically by
  [GoReleaser](https://goreleaser.com) on each tagged release — don't edit them by hand.
